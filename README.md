# Realtime-TTS-for-Home-Asistant
Bu add-on, Home Assistant'in intent sonunda uretilen yanit metnini alip secilen TTS saglayicisindan dusuk gecikmeli PCM akisi ceker ve Alice cihazi buna WebSocket ile baglanip sesi blok blok oynatir.  Desteklenen saglayicilar:  openai cartesia elevenlabs google_ai google_cloud
